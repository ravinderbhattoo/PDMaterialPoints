# Shapes

This page lists the shapes that are available in PDMesh. These shapes can be used to create meshes (see [`create`](@ref)), which can then be used for simulations. The following shapes are available:

- [`Cone`](@ref)
- [`Cuboid`](@ref)
- [`Cube`](@ref)
- [`Cylinder`](@ref)
- [`Disk`](@ref)
- [`Pyramid`](@ref)
- [`Indentor`](@ref)
- [`Sphere`](@ref)
